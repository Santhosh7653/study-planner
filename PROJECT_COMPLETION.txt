🚀 GET STARTED IN 3 MINUTES

1. Install dependencies:
   npm install

2. Configure Gmail (set in .env.local):
   GMAIL_USER=your@gmail.com
   GMAIL_PASS=16-char-app-password

3. Start servers:
   npm run dev:server        # Terminal 1 - Backend on :5000
   npm run dev:frontend      # Terminal 2 - Frontend on :5173

4. Open browser:
   http://localhost:5173

5. Create a task and watch the email arrive! 📧

📧 EMAIL SYSTEM - HOW IT WORKS

Automatic triggers:

✉️  Task Created
    When: User adds a new task
    Email: Task details with all info

✉️  Priority Changed
    When: Priority is updated
    Email: Shows old → new priority

✉️  Due Date Updated
    When: Deadline is modified  
    Email: Shows old → new date

✉️  Daily Summary
    When: 8:00 AM every day
    Email: All pending tasks for today

✉️  1-Hour Reminder
    When: 60 min before task deadline
    Email: Urgent reminder email

🆘 QUICK TROUBLESHOOTING

Problem: Dependencies won't install
Solution: npm cache clean --force && npm install

Problem: Port 5000 already in use
Solution: PORT=3001 npm run dev:server

Problem: Emails not sending
Solution: Check SETUP_GUIDE.md troubleshooting section

Problem: Frontend shows blank
Solution: Check DevTools Console (F12) for errors

Problem: Tasks not saving
Solution: Verify Firebase env vars in .env.local
